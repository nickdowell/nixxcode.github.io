BEARDYTRON 5000

The BEARDYTRON 5000 is a bespoke live performance music system built for Beardyman and compromises apps running on several iPads that control a custom Mac audio looper application that hosts various 3rd party effects.

I was brought onto this project to improve system performance as well as assisting with bug fixing and implementation of new features.
